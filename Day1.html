Задание 1.3

При отрисовке этого canvas-a были использованы кубические кривые Безье. Такая кривая может быть описана формулой $$x = (1-t)^2x_1 + 2(1-t)tx_2 + t^2x_3,$$ $$y = (1-t)^2y_1 + 2(1-t)ty_2 + t^2y_3,$$ где \((x,y)\) - координаты очередной точки кривой, \((x_i,y_i)\) - координаты \(i\)-й опорной точки, \(t\) - параметр, проходящий значения из диапазона \([0;1].\) Опорные точки берутся из множества точек контура.
